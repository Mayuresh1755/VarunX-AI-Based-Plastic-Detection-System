## 📌 Overview

**VarunX** is an AI-powered web application designed to detect and classify **macro-plastics** from images using deep learning techniques.

The system leverages **YOLO (You Only Look Once)** object detection to identify plastic waste and estimate its density within a defined area, enabling **environmental monitoring and analysis**.

---

## 🎯 Key Features

- 🧠 **YOLO-based Object Detection**  
  Detects and classifies plastic objects in images

- 📊 **Plastic Density Estimation**  
  Calculates the concentration of plastic within a region

- 🖼️ **Image Processing Pipeline**  
  Handles preprocessing, inference, and output generation

- 📈 **Visualization Output**  
  Displays bounding boxes and detection results clearly

## 🧠 Model Details

- **Model:** YOLO v12 (Object Detection)
- **Task:** Detect and classify macro-plastics
- **Approach:** Real-time single-stage detection
- **Output:** Bounding boxes + class labels

---

## 🔄 Workflow

1. Upload an image  
2. Preprocess the image  
3. Run YOLO model inference  
4. Detect plastic objects  
5. Calculate plastic density  
6. Display results
